* 关于本项目的情况
+ 可以点击 [[https://jansky520.github.io/][博客]] 进入我的个人博客阅读。但更推荐下载源码，使用 emacs 打开 .org 文件阅读。

+ 目前项目仍在缓慢推进，前期项目以直接翻译原文手册为主，但翻译效率低，大家阅读的也艰难，故后面准备以精修阅读笔记为主。
